# Gantt SaaS Practice

React와 Vite로 만든 연습용 간트차트 MVP입니다.

프로젝트 작업, 담당자, 시작일, 종료일, 상태를 입력하고 간단한 간트차트 형태로 확인할 수 있습니다.

## 주요 기능

- 작업 추가
- 작업 삭제
- 작업 목록 확인
- 간트차트 형태로 일정 시각화
- localStorage를 이용한 브라우저 저장

## 사용 기술

- React
- Vite
- JavaScript
- CSS
- Git / GitHub
- localStorage

## 실행 방법

npm install
